# CodePilot AI 🚀

AI-powered code review platform built with React, Node.js, Express, and Groq AI.

## Features

✅ AI Code Review
✅ Syntax Highlighting
✅ Markdown Rendering
✅ Multi-language Support
✅ Fast Responses using Groq
✅ Responsive UI

## Tech Stack

- React
- Tailwind CSS
- Node.js
- Express.js
- Groq API
- React Markdown
- Highlight.js

## Installation

### Client

npm install
npm run dev

### Server

npm install
npm run dev

Create a .env file:

GROQ_API_KEY=YOUR_API_KEY
PORT=5000
